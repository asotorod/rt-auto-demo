# RT Auto Center — Premium Dealership Website Demo

![React](https://img.shields.io/badge/React-19-blue) ![Tailwind](https://img.shields.io/badge/Tailwind-4-blue) ![Vite](https://img.shields.io/badge/Vite-7-purple)

A dark luxury dealership website demo for **RT Auto Center** (Newark, NJ). Built with React, Tailwind CSS v4, Framer Motion, and Vite. Features real inventory data and the client's actual logo.

## Pages

- **Home** — Hero with Bentley Continental, featured vehicles, value props, brand marquee
- **Inventory** — Full listing with search, make/body filters, sort options
- **Vehicle Detail** — Cinematic hero image, specs grid, payment estimator, inquiry form, similar vehicles
- **Financing** — Credit application with Capital One partner callout
- **About** — Company story, stats, core values
- **Contact** — Google Maps embed, contact form, business info

## Tech Stack

- React 19 + React Router 7
- Tailwind CSS v4 (custom theme with gold accent palette)
- Framer Motion animations
- Lucide React icons
- Vite 7 build
- Vercel deployment ready

## Design

- Dark luxury aesthetic (#050505 base, gold #c9a84c accents)
- Playfair Display + DM Sans typography
- Glass morphism navigation
- Scroll-triggered animations
- Fully responsive (mobile-first)

## Quick Start

```bash
npm install
npm run dev
```

## Deploy to Vercel

1. Go to [vercel.com/new](https://vercel.com/new)
2. Import the `rt-auto-demo` repo
3. Framework: Vite, Build: `npm run build`, Output: `dist`
4. Deploy!
